Dataset Sources

Our dataset combines multiple professional sources including the Saraga dataset (MTG), Harvard research collections, and curated YouTube recordings. Each raga includes multiple performances, scale definitions, characteristic phrases, and cultural context.

AI Models

YuE Foundation Model (2025)

State-of-the-art music foundation model adapted for Indian classical music

  • Multi-modal architecture (audio + text)
  • 95%+ classification accuracy
  • Support for 1,616+ ragas
  • Real-time inference capability
  • Cultural context understanding

Enhanced Temporal Encoder

Specialized architecture for complex Indian tala cycles

  • 32+ beat tala cycle support
  • Rhythm pattern recognition
  • Cycle detection algorithms
  • Long-range temporal modeling

Shruti Pitch Encoder

Microtonal pitch system for Indian classical music

  • 22-shruti system implementation
  • Raga-specific interval learning
  • Cultural pitch context
  • Melakarta system integration
